📋 Formato File Excel Richiesto

Il file Excel deve contenere le seguenti colonne (nell'ordine):

  1. INGRESSO LEAD - Data ingresso (formato: AAAA-MM-GG)
  2. Data ordine - Data ordine (formato: AAAA-MM-GG)
  3. SALES - Nome venditore
  4. Prodotto - ON LINE o VIRTUAL
  5. Membro / Referente - Nome e Cognome studente
  6. Prezzo - Importo numerico
  7. Rateizzato - Tipo di rateizzazione
  8. N CTR - Numero contratti
  9. INCASSO [MESE] - Importi incasso per ogni mese (Gennaio-Dicembre)

ℹ️ Il sistema leggerà il nome del foglio per determinare il mese di produzione (es: "LUGLIO 2025")
